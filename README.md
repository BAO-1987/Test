# Інструкція з запуску проекту

## Вимоги

Перед початком роботи переконайтеся, що у вас встановлено:
- Node.js (версія 18 або вище)
- npm або yarn

## Встановлення залежностей

Для встановлення всіх необхідних залежностей виконайте команду:

```bash
npm install
```

або

```bash
yarn install
```

## Запуск проекту

### Режим розробки (Development)

Для запуску проекту в режимі розробки з автоматичним оновленням при зміні файлів:

```bash
npm run dev
```

або

```bash
yarn dev
```

Після запуску проект буде доступний за адресою: `http://localhost:5173` (або інший порт, якщо 5173 зайнятий)

### Збірка для продакшену (Build)

Для створення оптимізованої збірки проекту:

```bash
npm run build
```

або

```bash
yarn build
```

Збірка буде створена в папці `dist/`

### Попередній перегляд збірки (Preview)

Для перегляду збірки перед деплоєм:

```bash
npm run preview
```

або

```bash
yarn preview
```

## Структура проекту

```
├── src/
│   ├── pages/          # Handlebars шаблони сторінок
│   ├── partials/       # Часткові шаблони (header, footer, тощо)
│   ├── scripts/        # JavaScript файли
│   └── styles/         # SCSS стилі
├── public/             # Статичні файли (зображення, іконки)
├── dist/               # Збірка проекту (створюється після build)
└── vite.config.js      # Конфігурація Vite
```

## Технології

- **Vite** - збірщик та dev-сервер
- **Handlebars** - шаблонізатор
- **SCSS** - препроцесор стилів
- **PostCSS** - обробка CSS з Autoprefixer
- **ViteImageOptimizer** - оптимізація зображень
- **ViteMinifyPlugin** - мініфікація HTML, CSS, JS

## Додаткова інформація

- Стилі перевіряються за допомогою Stylelint
- Код форматується за допомогою Prettier
- Зображення автоматично оптимізуються під час збірки
